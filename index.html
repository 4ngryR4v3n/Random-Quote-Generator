“[A] quotation is a handy thing to have about, saving one the trouble of thinking for oneself, always a laborious business.“

--A. A. Milne












This is a random quote generator completed as a project for Free Code Camp . Some of the parameters included providing a random quote and having the ability to tweet out that quote. I experimented with things on a new level and created three decks, all shuffled prior to dealing, and also having those decks appear in a random order with each deal. I wanted to ensure, as well, that all cards are dealt from a deck before re-shuffling and continuing to deal so the code ensures that happens even though the decks have different amounts of cards in each. My first version of the project had multiple instances of functions such as one instance each of a shuffle and deal functions for each deck (e.g. shuffleLeaderDeck(), shuffleLoveDeck(), and shuffleWiseDeck(). I modified it to pass variables from objects representing each deck into dealCard() and shuffleDeck() functions which cleaned up the code significantly and gave good practice working through those different solutions to be able to create scalable code easier in the future. Lastly, I explored different avenues of CSS including Bootstrap, Font-Awesome, and SASS to make it a responsive design.
